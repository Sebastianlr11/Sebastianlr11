**Desarrollador Full-Stack | Automatizaciones & IA | Trader Algoritmico**

Desarrollador de Colombia apasionado por crear soluciones que combinan tecnologia web, inteligencia artificial y automatizacion. Construyo desde aplicaciones web completas hasta agentes de IA y sistemas de trading algoritmico.

### Lo que hago

**Desarrollo Web Full-Stack**
- Aplicaciones web con React, TypeScript, Vite y Tailwind CSS
- Backend con Supabase (PostgreSQL), Edge Functions y autenticacion
- Integraciones de pago con Mercado Pago
- Deploy en Vercel

**Automatizaciones & Agentes de IA**
- Workflows inteligentes con n8n (self-hosted)
- Agentes conversacionales con Claude AI (Anthropic)
- Integracion con WhatsApp Business API e Instagram
- Function calling, vision, audio (speech-to-text)

**Trading Algoritmico**
- Expert Advisors y herramientas para MetaTrader 5 (MQL5)
- Analisis de portafolios y reportes automatizados
- Gestion de riesgo y optimizacion de estrategias
